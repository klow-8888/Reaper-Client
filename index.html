Твои Возможности

Skin & Clan Stealer

Присваивай идентичность жертв. Мгновенное извлечение скинов и кланов с безжалостной эффективностью.

Aliases System

Революционная система алиасов. Создавай собственные команды и макросы для мгновенного доступа к функционалу.

Player Dump

Полная разведка сервера. Получай всю информацию о цели: ID, имя, клан — всё для тотального доминирования.

Tile Out Notification

Уведомления о выходе с тайла на панели быстрого доступа.

Hide Chat Bubbles

Активируй режим тишины. Лиши противника последней надежды — возможности увидеть, что ты печатаешь в чат.

Old Star Particles

Наслаждайтесь ретро-стилистикой с классической системой частиц во фризе.

Fast Input

Твои реакции — это закон. Технология, стирающая задержку между мыслью и действием. Ты не успеваешь за клиентом. Ты — его часть.

Mini-Debug

Вскрой геймплей до костей. Мощные инструменты для анализа и оптимизации каждого твоего движения.

Tabbed Menu

Интуитивная навигация по вкладкам для мгновенного доступа к любому инструменту разрушения.

Improved Settings

Калибровка мощи. Точный ввод значений для тонкой настройки оружия под свою хватку. Никаких приблизительных ползунков.

Команды Reaper Client

Полный список команд для настройки клиента под себя

steal_skin

Скины

Копирует скин указанного игрока.

steal_skin [никнейм]
steal_skin PlayerName // Копирует скин игрока PlayerName

steal_clan

Скины

Копирует клан указанного игрока.

steal_clan [никнейм]
steal_clan PlayerName // Копирует клан игрока PlayerName

cl_hide_chat_bubbles

Визуал

Скрывает облачка чата над вами.

cl_hide_chat_bubbles [0|1]
cl_hide_chat_bubbles 1 // Включает скрытие облачков чата

fast_input

Геймплей

Активирует систему уменьшения визуальной задержки ввода для максимальной отзывчивости.

cl_fast_input [0|1]
cl_fast_input 1 // Включает быстрый ввод

alias

Алиасы

Создает новый алиас (псевдоним команды).

alias [имя] [команда]
alias say_hi "say hello!" // Создает алиас say_hi с командой "say hello"

unalias

Алиасы

Удаляет существующий алиас.

unalias [имя]
unalias say_hi // Удаляет алиас say_hi

list_aliases

Алиасы

Показывает список всех созданных алиасов.

list_aliases
list_aliases // Показывает все алиасы

alias_exec

Алиасы

Выполняет команду алиаса с дополнительными аргументами.

alias_exec [алиас]
alias_exec say_hi // Выполняет алиас say_hi

cl_mini_debug

Отладка

Включает компактный режим отладки для анализа игровых процессов.

cl_mini_debug [0|1]
mini_debug 1 // Включает мини-дебаг

player_dump

Отладка

Выводит информацию обо всех игроках на сервере.

player_dump
player_dump // Показывает информацию об игроках
